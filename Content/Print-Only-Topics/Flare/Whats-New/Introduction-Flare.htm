Introduction

Most of the new features in this release of Flare focus on HTML5 output. In many cases the emphasis is on Top Navigation output in particular.

Feature

What's New?

Google Search

  • Alternative to MadCap Search
  • Set up search engine with Google search key
  • Beneficial especially for search on mobile devices

Favicons

  • “Favorite” icons
  • Select custom image for different areas of browser (e.g., tab, bookmark, Windows tiles) and smart phone shortcuts
  • Add one large image and Flare automatically resizes it for different uses

Fixed Headers

  • Fix header area in HTML Top Navigation output
  • Remains in view as users scroll down

Fixed Topic Menus

  • Fix side topic menu in HTML Top Navigation output
  • Remains in view as users scroll down

Heading Option for Topic Menus

  • Display headings and subheadings in side topic menus
  • Alternative to displaying related topics

Tree Format for Slide-Out Menus

  • Alternative to drilldown look of slide-out menu in Top Navigation output
  • Click arrows to display sub-items; rest of menu remains in view

Custom Installation

  • Choose to include local Help files
  • Choose to enable Salesforce integration

Salesforce Integration

  • Integrate Flare with your Salesforce account
  • Use destination file to publish output directly to Salesforce

FrameMaker 2017 Support

  • Import files from FrameMaker 2017

Synchronizing Navigation Elements with TOC Entries

  • Keep navigation elements (e.g., menu items, breadcrumbs, mini-TOCs) synchronized with TOC
  • Useful when TOC contains multiple entries linking to same topic
