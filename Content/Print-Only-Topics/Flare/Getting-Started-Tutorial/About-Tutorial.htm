Using the Getting Started Tutorial

The Getting Started Tutorial covers the following:

  1. Starting Projects How to build a new project from a template. See Starting Projects.
  2. Adding Elements to Projects How to add content to your projects, such as topics, images, cross-references, and snippets. See Adding Elements to a Project.
  3. Making It Look Good How to use styles to make your project look the way you want it to. See Making It Look Good.
  4. Developing Targets How to prepare your output using conditions, PDF and HTML5 outputs, skins, and tables of contents. See Developing Targets.
  5. Building Output How to build the final printed and web outputs for your project. See Building Output.
