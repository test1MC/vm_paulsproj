Primary Window Panes

has numerous window panes in the interface that are used for a variety of purposes. Some elements are contained in window panes as opposed to dialog windows because they contain features that you would want to have easy access to as you work in an editor.

Following are the window panes that are probably most commonly used in :

  • Effect Properties Window Pane
  • Frame Properties Window Pane
  • Keyframe Properties Window Pane
  • Object Properties Window Pane
  • Sound Properties Window Pane
  • Timeline Window Pane
