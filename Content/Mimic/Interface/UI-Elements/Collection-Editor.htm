Collection Editor

You can create standalone movies or movie collections. The Collection Editor is used as a central control area for all of the movies contained within a collection. The Collection Editor has a local toolbar, allowing you to easily add movies to your collection. There are also multiple tabs in the editor, used for various purposes.

Collection Editor Local Toolbar

You can use the local toolbar to manage your movie collection directly from the Collection Editor.

  • Movies Window Pane (Open) See Movies Window Pane.
  • Blank movies (Create) See Creating Blank Movies.
  • Import Mimic Movies See Importing Movies.
  • Import PowerPoint Presentations See Creating Movies from PowerPoint Presentations.
 Sections

The Sections tab allows you to modify movie and collection details, such reordering the movies within the collection and renaming movies and files.

  • Rename Movie Files See Renaming Movies.
  • Change Title See Adding Titles to Movies.
  • Open Movies See Opening Movies.
  • Delete Movies See Deleting Movies.
  • Organize Movies See Organizing Movies in a Collection.
 Options

The Options tab includes output preferences.

  • Output Format (Select) See Selecting the Default Movie Format.
  • Single Files See Selecting Single Files for the Output.
  • Comments (Add) See Adding Comments to Collections.
 Language

  • Language for Collection (Select) See Selecting a Language.
 Variables

You can create variables for collections. The variables you create here are available to all movies within the collection.

  • Variables (Create) See Creating Variables.
 Condition Tags

You can create your own condition tags. Condition tags created here are available to all movies within the collection.

  • Condition Tags (Create) See Creating Condition Tags.
 Conditions

After applying condition tags to the appropriate elements, you can tell Mimic to include or exclude those conditions in the output.

  • Condition Tags (Associate) See Associating Condition Tags.
 Targets

You can create as many targets as you need. You can use different settings for each target, so that each output is unique.

  • Targets (Create) See Creating Targets.
  • Targets (Edit) See Editing Targets.
 Project Link

Linking a collection to a Flare project allows you to automatically have access to any variables or condition tags found in that Flare project.

  • Flare Project Links See Linking to Flare Projects.
