Windows Server 2012 Requirements

Pulse Requirements for Windows Server 2012

As you prepare for your Pulse installation, familiarize yourself with the minimum system requirements.

Pulse Web Server

pulse web server

The Pulse web server provides users the ability to access and use the web-based user interfaces of the Pulse system.

web server—minimum hardware requirements web server—operating system requirements

This table details the operating system requirements to install the Pulse web server:

All of the latest Windows service packs and critical updates must also be installed.

Requirement

Notes

Microsoft Windows Installer (MSI) 5.0

The MSI 5.0 is included with the operating system.

Internet Information Services Version 8

This is included with the operating system. Use the Server Manager to add the Web Server (IIS) role service. See How to Install IIS.

ASP.NET 4.5

This is included with the operating system. Use the Server Manager to add the ASP.NET role service. This is covered in the steps for installing IIS.

Microsoft .NET Framework 4.5

This is included with the operating system and may already be installed on your server, typically by a Network Administrator. If it is not installed, you can download it from:

http://www.microsoft.com/en-us/download/details.aspx?id=30653

web server—pulse setup program

The Pulse setup program installs the Pulse application on the web server (this is the server running IIS). Before running the setup program, you must make sure Microsoft .NET Framework 4.5 is installed (see Installing the Prerequisites for Windows Server 2012).

Pulse Database Server

pulse database server
