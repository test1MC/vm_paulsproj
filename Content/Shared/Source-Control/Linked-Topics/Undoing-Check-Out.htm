Reverting or Undoing a Checkout of Source Control Files

You can revert or undo a checkout, commit, or modification of a source control file in the following ways:

  • Apache Subversion Reverting Modified Source Control Files
  • Git Reverting Modified Source Control Files
  • Microsoft Team Foundation Server Undoing a Checkout of Source Control Files
  • Microsoft Visual SourceSafe Undoing a Checkout of Source Control Files
  • Perforce Reverting a Checkout of Source Control Files
