Deleting Branches—Git

You can delete any branch except the current branch. If you need to delete the current branch, you will have to switch to a different branch first. See Switching Branches—Git.

how to Delete Branches—Branch Management Dialog

    • Ribbon In the Project section select Branch.

    • Right-Click If you have the Content Explorer, Project Organizer, Pending Changes window pane, or File List open, right-click on any file and select Source Control > Project > Branch.
    • Right-Click If you have the File List open, right-click on any file and select Source Control > Project > Branch.

    The Branch Management dialog opens.

  2. In the All branches list, select the branch you want to delete. You can delete any branch except the current branch.

    Your current branch is shown at the bottom of the dialog.

  3. Click Delete.

  4. A dialog asks if you want to delete the branch and all of the commits on the branch. Click Yes. The branch is removed from the list.

 how to Delete Branches—Source Control Explorer
  1. In the Explorer section select Source Control Explorer. The Source Control Explorer opens.
  2. In the Pane section select Source Control Explorer. The Source Control Explorer opens.

  3. From the drop-down or the Home pane, select Branches.

    The Branches pane opens. In the pane, you can see your current branch, as well as lists of your published and unpublished branches.

  4. Select the branch you want to delete. You can delete any branch except the current branch.

  5. Click Delete Branch.

  6. A dialog asks if you want to delete the branch and all of the commits on the branch. Click Yes. The branch is removed from the list.
