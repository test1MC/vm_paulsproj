Disabling the Get Latest Prompt for Source Control—Git

how to disable the get latest prompt for source control
  1. Select File > Options.

    The Options dialog opens.

  2. Select the Source Control tab.

    This tab will not be visible if your project is not yet bound to source control. See Binding a Project to Git.See Binding a Project to Git.

  3. Click the check box Do not prompt to get latest when opening source control bound projects.
