Deleting Source Control Files—Perforce

You can delete a topic or file that is bound to source control. This also removes the file from Perforce.

how to delete a File
  1. In one of the window panes (e.g., Content Explorer, File List, Project Organizer, Pending Changes window pane), select the relevant file(s).
  3. The Delete dialog opens. Select the bound files you want to delete.
  4. Click Delete. The files are removed from your project and from the source control repository.
 What's Next?

You can undelete a file from Perforce if you need to recover it. See Undeleting Source Control Files—Perforce.
