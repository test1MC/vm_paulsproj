Manage Translation Memory Dialog

This dialog lets you work with translation memories (TMs) in the following ways.

  • Add TMs You can add an existing TM to your project using this dialog.
  • Create TMs You can create a new, empty TM using this dialog.
  • Remove TMs If you are no longer using a TM, you can remove it.
  • Delete TMs If you no longer need a TM or the data in the TM, you can delete it.
  • Import TMX Files If you have created a TM in , you can import a TM file into it. You an import Translation Memory eXchange (TMX). .
  • Export TMX Files If you have created a TM in , you can export it to a Translation Memory eXchange (TMX) file.
