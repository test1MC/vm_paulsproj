Manage Termbases Dialog

This dialog lets you work with termbases in the following ways.

  • Add Termbases You can add an existing termbase to your project using this dialog. See Adding Termbases.
  • Create Termbases You can create a new, empty termbase using this dialog. See Creating Termbases.
  • Edit Terms See Editing Termbases.
  • Remove Termbases If you are no longer using a termbase, you can remove it. See Removing Termbases.
  • Delete Termbases If you no longer need a termbase or the data in the termbase, you can delete it. See Deleting Termbases.
  • Import TBX Files If you have created a termbase in , you can import a termbase file into it. You an import TermBase eXchange (TBX), CSV, XLS, See Importing TBX Files.
  • Export TBX Files If you have created a termbase in , you can export it to a TermBase eXchange (TBX) file. See Exporting TBX Files.
