Manage Translation Memory Dialog

This dialog lets you work with translation memories (TMs) in the following ways.

  • Add TMs You can add an existing TM to your project using this dialog. See Adding a Translation Memory.
  • Create TMs You can create a new, empty TM using this dialog. See Creating a Translation Memory Database.
  • Edit TMs You can open your TM database and edit or filter it in various ways. See Editing the Translation Memory Database.
  • Remove TMs If you are no longer using a TM, you can remove it. See Removing a Translation Memory.
  • Delete TMs If you no longer need a TM or the data in the TM, you can delete it. See Deleting a Translation Memory.
  • Import TMX Files If you have created a TM in , you can import a TM file into it. You an import Translation Memory eXchange (TMX). See Importing TMX Files.
  • Export TMX Files If you have created a TM in , you can export it to a Translation Memory eXchange (TMX) file. See Exporting TMX Files.
