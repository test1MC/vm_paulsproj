Specifying Quick Characters

how to specify a quick character
  1. Open a file in the Translation Editor.
  2. (Optional) If the character is shown in the file you have open, you can select and copy it.
    • Ribbon In the Symbols section, click the face of the Character button.
    • Keyboard Shortcut Press SHIFT+F11.

    The Character dialog opens.

    • Click in the Quick Character field, removing the previous character if necessary. Then paste your selection into that field.
    • Right-click on any character shown in the dialog, and from the context menu select Set as quick character.
    • Click once on any character shown in the dialog, and on the right side of the dialog click .
  5. Click Close.
