Translation

Lingo offers several features that help you translate a project, including an alignment tool, machine translation, multilingual options, translation memories, and segmentation rules.

Translation Features

Here are the primary features to help you translate a project:

  • Alignment See About Alignment.
  • Bundles See About Bundles.
  • Concordance See About the Concordance.
  • File Filters See About File Filters.
  • Machine Translation Machine translation (MT) is translation of text by a computer or translation program, such as Google Translate. A great benefit of MT is that it allows translators to translate a large amount of content quickly using an automated translation system, and then make any necessary updates to the translation on a case-by-case basis (e.g., using a termbase to replace certain words, fixing translations that are not accurate in context). Then, the translator can add the target segments to an installed translation memory (TM), improving the speed for future translations as well. See About Machine Translation.
  • Multilingual See About Multilingual Projects.
  • Reports When working with translations in , you can generate statistics and quality assurance reports to see the status of your translation project. See Reports.
  • Termbases See About Termbases.
  • Translating Files See Translating Files.
  • Translation Memory See About Translation Memory.
  • Segmentation Rules If you want to control segment breaks automatically, you can create segmentation rules. follows the format specified by Segmentation Rules eXchange (SRX). Therefore, after you create segmentation rules, you can export them to an SRX file. You can also import SRX files that have been generated in . See About Segmentation Rules.
