Inserting Pages in Page Layouts

There are various ways to add new pages to a page layout.   See About Page Layouts.

how to insert a page into a page layout

A new untitled page is added to the page layout. A small rectangle is added to the right of the page layout, which can be used to open the new page.

 note

You can also mirror, duplicate, or copy a page. See Mirroring Pages in Page Layouts, Duplicating Pages in Page Layouts, and Copying Pages in Page Layouts.
