Adding Auto Suggestion List Files

how to add an auto suggestion list file
    • Ribbon In the Content section select New > Advanced > Auto Suggestion List File.
    • Right-click In the Project Organizer, right-click on the Advanced folder and from the context menu select Add Auto Suggestion List File.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Auto Suggestion List File is selected.

  3. The auto suggestion list file supports the TBX-Basic markup format, and its extension is .fltbx (e.g., MyAutoSugList.fltbx).

  5. In the File Name field, type a new name for the auto suggestion list file.
  6. (Optional) See Applying Conditions to Content.
  7. (Optional) See Associating Tags with Files.
  8. Click Add. The auto suggestion list file is added to the Advanced folder in the Project Organizer. The Auto Suggestion List Editor opens to the right.
 What's Next?

After you add an auto suggestion list file to your project, you can add words, phrases, or sentences (256 character limit) to it. You can do this directly in the Auto Suggestion List Editor, or you can right-click text in the XML Editor and use the context menu to add it to the auto suggestions list file. See Creating Auto Suggestion Lists.
