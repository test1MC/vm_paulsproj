Searching an HTML5 System

End users have a variety of options when performing a search in a system.

Search Experience

search experience

An skin (see About Skins) provides the user interface for an system. When an end user searches an system for information, it reveals a list of topics containing the matching term or phrase. The search results show the topic title and the topics are listed in ranked order. The results also include an excerpt of the paragraph text from the topic (or you can opt to use the meta description that you add to the topic properties). This helps users decide if the page contains the information they were looking for. When a user clicks a topic to open it, the matching terms are highlighted in the topic.

How to Search

how to search

  1. Open the  system.

  2. In the search field, type the search query. Then click the Search button or press Enter.

  3. In the search results list, double-click one of the topics to open it. Search highlighting reveals the matches in the topic.
