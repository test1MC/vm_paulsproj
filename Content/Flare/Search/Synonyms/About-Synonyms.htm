Tasks for Using Synonyms

tasks for using synonyms

Following are the primary tasks in for using synonyms:

  • Add Synonym File The first step to creating synonyms for enhanced search is to add a synonym file to the Flare project. See Adding Synonym Files.
  • Create Synonyms After adding a synonym file to your project, you can create synonyms within it. See Creating Synonyms to Enhance Search Results.
  • Export Synonym Files If you are using Feedback and have created synonyms, you may want to export the synonyms file (MCSYNS) to another location. Those synonyms can then be added to a different Flare project. See Exporting Synonyms.
  • Open Synonym Files After a synonym file is created, you can open it at any time to make adjustments to it. See Opening Synonym Files.
 notes

Depending on the browser being used, the cache may need to be cleared in order to see the synonym results.
