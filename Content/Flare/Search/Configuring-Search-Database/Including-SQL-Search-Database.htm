Including an SQL Search Database

You have the option to include an SQL database to support search with the published output. This can improve search performance and permit users to use the asterisk (*) wildcard with search. For example, if you type Mad* in the Search field, will list all topics with words that have "Mad" followed by any other letters (e.g., MadCap, Madagascar, madness, Madrid).

how to include an sql search database
  1. In the Project Organizer, open the target.
  2. Select the Performance tab.
  3. In the Search Database section, place a check mark in the Include SQL Search Database box.
