Adding an MS Word Import File

The following steps show you how to add an MS Word import file, which is used to import Word documents into a project.

how to add an ms word import file to a project
    • Ribbon In the Content section select New > Import File > Add MS Word Import File.
    • Right-Click In the Project Organizer, right-click on the Imports folder and from the context menu select Add MS Word Import File.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure MS Word Import File is selected.
  5. In the File Name field, type a new name for the import file.
  6. (Optional) See Applying Conditions to Content.
  7. (Optional) See Associating Tags with Files.
  8. Click Add. The import file is added to the Imports folder in the Project Organizer. The Word Import Editor opens to the right.
 note What's Next?

After you add the new import file, you can import Word documents into the project. See Importing Word Files.
