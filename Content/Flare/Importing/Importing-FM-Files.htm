How to Import FrameMaker Files

  1. Do the following, depending on how you are importing your files:

    Import into a New Project
    1. Select File > New Project > FrameMaker Documents.

      The Import FrameMaker Documents Wizard opens.

    1. Select Project > Import > FrameMaker Documents.

      The Import FrameMaker Documents Wizard opens.

    1. Complete the steps for adding an import file. See Adding a FrameMaker Import File.

    if you already have a framemaker import file

    1. In the Project Organizer, expand the Imports subfolder.
    2. Double-click the appropriate import file. The file opens in the Frame Import Editor to the right.
