Text Box Properties Dialog

This dialog is used to set properties for a text box. Following are the primary tasks that you can perform in this dialog. For more information see About Text Boxes, Inserting Text Boxes, and Editing Text Boxes.

  • Background See Adding Backgrounds to Text Boxes.
  • Borders See Adding Borders to Text Boxes.
  • Hyphenation You can adjust the way that the text within a text box is hyphenated. See Setting Hyphenation in Text Boxes.
  • Margins See Adding Margins to Text Boxes.

  • Padding See Adding Padding to Text Boxes.

  • Position You can adjust the positioning of a text box on a page. This includes the ability to float text boxes to the left or right, or even outside of page frames. See Using Floats for Positioning Elements.
  • Size You can adjust the width and height of a text box. See Specifying the Size of Text Boxes.
