Button/Section

Description

Template Folders

Lists the folders containing templates for the type of item you are working with. The "Factory Templates" folder holds templates that are provided by . The other folders hold your own customized templates that you can create. When you select one of the folders, the available templates are listed to the right in the "Templates" section.

Templates

Lists the templates associated with the folder selected to the left. What are templates?  For more information see About Templates.

Following are special templates provided by :

  • Running Header 1.flsnp Inserts a snippet containing a "Level 1" Heading variable, which displays the first "h1" text on the page (i.e., if you have applied the h1 style to heading text). You might use this snippet, for example, if you have used the h1 style throughout your project to begin new chapters. Therefore, when you use this snippet, the chapter title would be shown in the frame. See About Variables.
  • [This page intentionally left blank].flsnp Displays the text "[This page intentionally left blank]."
tip

This is also true for the other snippets that you insert. Once they are inserted, you can use the Frame Contents window pane to modify them.

Preview

Source File

Displays the path of the template file after you select it in the Templates section. You can also click the browse button to search for file that you want to import.
