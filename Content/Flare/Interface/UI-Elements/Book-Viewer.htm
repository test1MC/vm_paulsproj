Book Viewer

This viewer can be used to display native XHTML content generated from a target. From this viewer, you can navigate to different parts of the output and send the content to a printer. The buttons and features that you see in the viewer depend on the output type. For example, the XHTML output shows the page numbers in the lower-right corner of the page, as well as fields to quickly jump to a particular page. See About XHTML Output.
