Snippet Variables Dialog

You can perform the following actions from this dialog:

  • Set Snippet Variable Definitions Set the definitions you want to use for variables in the selected snippet. See Overriding Variable Definitions in Snippets.
  • Edit Variables Open the Variable Set Editor, where you can create new variables or edit existing variables. See Editing Variables.
Examples Note

If you want to override snippet variables at the topic level, you can do so from the Topic Properties dialog. See Properties Dialog (Topics).
