Primary Window Panes

has numerous window panes in the interface that are used for a variety of purposes. The two panes that you are likely to use most are the Project Organizer and Content Explorer. Some elements are contained in window panes as opposed to dialog windows because they contain features that you would want to have easy access to as you work in an editor.

Following are the window panes that are probably most commonly used in :

  • Builds Window Pane
  • Content Explorer
  • File List Window Pane
  • Project Organizer
  • Styles Window Pane
