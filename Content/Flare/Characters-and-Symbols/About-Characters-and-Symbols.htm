About Characters and Symbols

You can work with characters and symbols in in the following ways:

  • Inserting   See Inserting Characters and Symbols.
  • Quick Characters See Specifying Quick Characters.
  • Font Types See Character Dialog Font Types.

  • Smart Quotes If your project is using a language that does not use straight quotation marks, replaces quotes based on your project's current language. Additionally, if you are using a different language in a specific topic or paragraph, will use the correct smart quote for that language setting. See Using Smart Quotes.
