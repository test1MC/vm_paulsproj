Advanced Conversion to Top Navigation Output

The process takes longer than a simple conversion, but you end up with a project that has some enhanced designs, such as a full-featured Home page and a formatted context-sensitive menu on the side of topics.

If instead you would like to convert your project to Top Navigation output using a simpler process, see Simple Conversion to Top Navigation Output.
