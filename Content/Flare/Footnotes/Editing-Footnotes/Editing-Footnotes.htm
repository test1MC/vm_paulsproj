footnote editing tasks 

Following are some of the more common footnote editing tasks.

  • Border   See Editing the Border Above Footnotes.
  • Comment Font Settings   See Editing Footnote Comment Font Settings.
  • Comment Position   See Editing Footnote Comment Position.
  • Comment Spacing   See Editing Footnote Comment Spacing.
  • Comment Text   See Editing Footnote Comment Text.
  • Number Format or Symbol   See Editing Footnote Number Formats.
  • Number—Look   See Editing the Look of Footnote Numbers.
  • Number Restart   See Restarting Footnote Numbering.

In many cases, you can use either styles or local formatting to make changes to footnotes. It is recommended that you use styles whenever possible.

 footnote styles

The following styles can be used for editing footnotes.

  • MadCap|footnote 
  • MadCap|footnoteBlock 
  • MadCap|footnotesBlock 
  • MadCap|endnoteBlock 
  • MadCap|endnotesBlocks 

For more information about using styles, see About Styles and Stylesheets.
