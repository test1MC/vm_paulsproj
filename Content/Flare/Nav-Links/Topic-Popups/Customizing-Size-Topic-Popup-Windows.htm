Customizing the Size of Topic Popups

If you insert a topic popup link, the popup displays in a default window size in the output (after the link is clicked). However, you can adjust the style to specify a custom size for the popup box. You can do this by selecting the a.Popup style class and changing the values for the mc-popup-width and mc-popup-height properties (located in the Unclassified property group).
