About Web Toolbars

Creating a web toolbar is simply a matter of adding and editing the appropriate skin, depending on the output type. As long as that skin is associated with your target, end users will see the web toolbar in the output.

  • HTML5 Tripane See Web Toolbars in HTML5 Output.
  • WebHelp, WebHelp Plus, WebHelp Air See Web Toolbars in WebHelp Outputs.
  • HTML Help See Web Toolbars in HTML Help Output.
