Adding Destination Files

The first step in publishing output is to add a destination file to your project.

A destination file has an .fldes extension and is stored in the Project Organizer under the Destinations folder.

how to add a destination file to a project
    • Ribbon In the Content section select New > Destination.
    • Right-Click In the Project Organizer, right-click on the Destinations folder and from the context menu select Add Destination.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Destination is selected.
  5. In the File Name field, type a new name for the destination.
  6. (Optional) See Applying Conditions to Content.
  7. (Optional) See Associating Tags with Files.
  8. Click Add. The destination is added to the Destinations folder in the Project Organizer. The Destination Editor opens to the right.
 What's Next?

After you add the new destination file, you need to create the publishing destination by providing the necessary details (host, directory, etc.). See Creating Publishing Destinations.
