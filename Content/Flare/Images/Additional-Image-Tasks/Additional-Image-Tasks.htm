Additional Image Tasks

In addition to the primary tasks you can perform related to images, you can also do the following:

  • About Box—Selecting an Image See Selecting an Image for the About Box.
  • Deleting—Files from Project Image files are added to the project automatically when you insert an image into a topic. However, you can delete them at any time. See Deleting Image Files from the Project.
  • Deleting—Images from Content Files You can delete an image that you previously inserted into a content file. See Deleting Images from Content Files.
  • Image Hyperlinks—Inserting See Inserting Image Hyperlinks.
  • Image Hyperlinks—Editing See Editing Image Hyperlinks.
  • Image Maps—Creating See Creating Image Maps.
  • List of Images—Creating See Creating a List of Images.
  • Opening See Opening Images.
  • Page Layout Frames—Images See Adding Images to Page Layout Frames.
  • Pasting—Images from Other Programs See Pasting Images from Other Programs.
  • Regenerating You can quickly regenerate (update) all of the images in your project. See Regenerating Images.
  • Removing—Images from Headings in a Generated TOC See Removing Images from Headings in a Generated TOC.
  • Removing—Unreferenced Embedded Images from Skins See Removing Unreferenced Embedded Images from Skins.
  • Screen Captures—Adding to Projects See Adding Screen Capture Files to Projects.
  • Screen Captures—Inserting See Inserting Screen Capture Images.
  • Web Images—Showing and Hiding You can link image tags to websites rather than to images in your project. However, this may slow down your project, so you might want to disable the rendering of such images in the XML Editor. See Showing and Hiding Web Images in the XML Editor.
  • Web-Safe Images See Generating Web-Safe Images.
  • Word Output—Embedding Images An option on the Advanced tab of the Target Editor lets you ensure that images are embedded in output. See Embedding Images in Microsoft Word Output.
