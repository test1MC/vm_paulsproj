Adding an Export Project File

The following steps show you how to add an export project file, which is used to export projects.

how to add an export project file
  1. In the Content section select New > Add Export Project File. The Add File dialog opens.
  2. In the File Type field at the top, make sure Export Project File is selected.
  4. In the File Name field, type a new name for the export file.
  5. (Optional) See Applying Conditions to Content.
  6. (Optional) See Associating Tags with Files.
  7. Click Add. The export file is added to the Exports folder in the Project Organizer. The Export Project File Editor opens to the right.
 What's Next?

After you add the new export file, you can export the project or part of it. See Exporting Projects.
