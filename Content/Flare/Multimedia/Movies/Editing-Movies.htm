Editing Movies

You can edit movies in a variety of ways, depending on how you inserted the movie and what you want to edit. You can do any of the following.

  • Edit Flash, Windows Media, Quicktime, WebM, YouTube, or Vimeo Movie Properties If you have inserted an embedded Flash, Windows Media, Quicktime, WebM, YouTube, or Vimeo movie, you can edit the properties for it.
  • Edit Mimic Movies If you have inserted a link to a MadCap Mimic movie, you can open the movie in Mimic, where you can make enhancements and changes to it.
  • Edit Mimic Movie Link Properties If you have inserted a link to a MadCap Mimic movie, you can edit the properties of the link.
