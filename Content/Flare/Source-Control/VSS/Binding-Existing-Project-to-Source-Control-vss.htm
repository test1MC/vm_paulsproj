Binding a Project to Microsoft Visual SourceSafe

Use the following steps if you have already created a project and want to bind ("connect") it to Microsoft Visual SourceSafe.

how to bind a project to source control
  1. Open the project.
  2. In the Properties section select Project Properties. The Project Properties dialog opens.
  15. In the Project Properties dialog, click OK. The project is connected to Visual SourceSafe, and you can now check files in and out as necessary.
 how to Bind a Project to source control—Source Control Explorer
  1. Open the project.
  2. In the Explorer section select Source Control Explorer. The Source Control Explorer opens.

  14. The project is connected to Visual SourceSafe, and you can now check files in and out as necessary.
 note

You can also bind a new Flare project to source control while creating it. See Creating a New Project.
