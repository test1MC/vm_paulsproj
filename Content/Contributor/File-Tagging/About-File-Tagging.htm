About File Tagging

example

example

Let's say an author creates tags in Flare in order to keep track of the status of files (e.g., In Progress, Sent for Review, Completed). The author then sends a topic to you for review, with the request that you mark the file as completed after you are finished.

With the topic open, you select File > Properties. This opens the Properties dialog for that file. In that dialog you select the File Tags tab and choose the appropriate option.

You can then return the topic to the author, who adds it to the Flare project.

 steps for using file tagging

Following are the basic tasks involved with using the file tagging feature in :

  • Associate Tags with Files You can associate file tags with the appropriate files. If you associate certain tags with a folder, all of the files in that folder will be associated with those tags. See Associating Tags with Files.
  • Edit File Tag Sets You might need to edit a file tag set from time to time. For example, if you have a file tag set designed to hold author names and you hire a new writer, you need to open the file tag set and add that author's name. You also might decide to change the tag type (e.g., change "Author" to "Contributor"). See Editing File Tag Sets.
 note

These tags are intended to be assigned to entire files, not to paragraphs or other content within a topic.
