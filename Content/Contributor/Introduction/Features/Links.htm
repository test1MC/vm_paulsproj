Links

provides many ways to add links to your project. Links allow you to point to additional information in the same topic, from a different topic, or from a file outside of the project. These include the following:

  • Bookmarks See Inserting Bookmarks.
  • Cross-References See About Cross-References.
  • External Links See Inserting Links to External Files—HTML, PDF, Microsoft Office.
  • Named Destinations See Creating Named Destinations.
