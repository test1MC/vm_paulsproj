Viewing Footnotes in Topics

 

how to view all footnotes for a topic
  1. Open a topic.

  2. In the Print Layout section, select Footnotes.

    The Footnotes window pane opens, listing each footnote in the topic, with information about the position, sequence, and comment text of each. You can also change the position or sequence of any of the footnotes by using the drop-down fields in the local toolbar.
