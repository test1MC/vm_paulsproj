Inserting 3D Models

If you want to enhance your output using 3D, supports Universal 3D (U3D) files. These files allow you to show users an interactive 3D model in print or web output.

  3. Select Insert > Multimedia. Select any option from the drop-down. The Insert Multimedia dialog opens.

    note

    There is not a dedicated 3D Model option to open the Insert Multimedia dialog in Contributor, but you can still insert 3D models from this dialog.
  4. Select the General tab.
  5. Click to find and select a file to insert.
  6. In the dialog that opens, select 3D Model Files from the Files of type drop-down, then find the file you want to use.

  8. (Optional) If you want to apply a specific style class to the 3D model, you can select it from the Style Class field. For more information see About Styles and Stylesheets.
