About Characters and Symbols

You can work with characters and symbols in in the following ways:

  • Inserting   See Inserting Characters and Symbols.
  • Quick Characters See Specifying Quick Characters.
  • Font Types See Character Dialog Font Types.
  • Smart Quotes By default, adds straight quotation marks ("). However, you can select an option to automatically use single and double curly smart quotes (") instead when you work in the editor. This option is disabled by default. See Using Smart Quotes.
