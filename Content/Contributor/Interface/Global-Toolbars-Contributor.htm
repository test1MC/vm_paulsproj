Global Toolbars

Global toolbars can be seen only if you are working in the traditional tool strip view, as opposed to the ribbon view.

standard toolbar

 

Description

Opens the Start Page, which provides links to high-level tasks and information. See Start Page.

Opens the New Document dialog, which you can use to create a new document. See Creating New Documents.

text format toolbar

 

Description

See About Redacted Text and Setting Redacted Text on Content.

Lets you select one of several kinds of bullet or numbered lists, applying it to the selected content.

  • Custom Format Opens a custom list format that lets you create a uniquely styled list up to 10 levels deep. You can also select an existing custom format from this submenu to apply it to the selected content in your topic or snippet. See Creating Custom List Formats.

review toolbar

 

Description

Opens the Review Packages window pane, which displays all of the topic files contained in a review package received from an author. From this window pane you can open the topics to review and edit them. See Steps for Reviewing Documents and Review Packages Window Pane.

Note

For Lingo review package files, some options in the toolbars are disabled.
