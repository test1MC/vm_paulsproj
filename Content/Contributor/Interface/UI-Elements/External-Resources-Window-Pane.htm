External Resources Window Pane

The External Resources window pane lets you select and maintain groups of external files in one place. External resources can be virtually any local or network files that you have access to (e.g., images, PDF files, project files). You might use this feature if there are certain reference files that are related to the documents you are working on in .

  • In the Resources section select External Resources.
