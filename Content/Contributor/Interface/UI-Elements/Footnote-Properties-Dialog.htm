Footnote Properties Dialog

You can use this dialog to accomplish several editing tasks for footnotes, including the following.

Tasks Associated with Editing Footnotes
  • Comment Font Settings (Font Tab) See Editing Footnote Comment Font Settings.
  • Comment Position (Footnote Tab, Position Field) See Editing Footnote Comment Position.
  • Comment Spacing (Advanced Tab) See Editing Footnote Comment Spacing.
  • Number Format or Symbol (Footnote Tab, Format Fields) See Editing Footnote Number Formats.
  • Number—Look (Footnote Tab, Style CSS Span Class Fields) See Editing the Look of Footnote Numbers.
  • Number Restart (Footnote Tab, Number Field) See Restarting Footnote Numbering.

For more information and editing tasks, see About Footnotes and Editing Footnotes.

You can also use this dialog to edit various font properties for footnotes. See Editing Font Properties.
