    if using advanced view
    3. From the area below, select the img tag. If you created classes of the tag, you can expand the img style and select the class instead.

    7. Locate the padding property that you want to change. Each side (bottom, left, right, top) has separate properties that you can set (e.g., padding-top). If you plan to have the same settings for all four sides, you can simply use the padding property.
how to add padding to images using local formatting
  1. Select the Borders & Margins tab.

  2. Set the options in the Padding section.
