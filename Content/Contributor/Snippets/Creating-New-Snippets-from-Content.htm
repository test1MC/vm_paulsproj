Creating New Snippets from Existing Content

Snippets are pre-set chunks of formatted content that you can use over and over.

The following steps show you how to create new snippets from existing content.

how to create a new snippet from existing content
  1. Open the document.
  2. In the XML Editor, highlight the content that you want to turn into a snippet.

  3. In the Snippet section select Create Snippet.

  4. In the Save As dialog, provide a name for the snippet and navigate to the location where you want to store it.
  5. Click Save.
