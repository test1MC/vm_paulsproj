Adding Folders to External Resources

You can browse and select local and network folders containing files that you want to have easy access to.

how to add folders to the external resources window pane
  1. In the Resources section, select External Resources. The External Resources window pane opens.
  2. In the local toolbar, click .
  3. In the Select Folder dialog, find and select the folder that you want to add.
  4. The folder is added to the window pane.
 What's Next?

After you add folders to the External Resources window pane, you can open the files within them.
