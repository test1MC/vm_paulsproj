Navigation Panes (Word)

Doc-To-Help navigation panes in Word make it possible to access Doc-To-Help functionality without having to switch back and forth to the Doc-To-Help application. You can use these panes to build and view output from your targets; see build status; and work with source documents, topics, keywords, and variables.
