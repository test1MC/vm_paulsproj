Ribbons

Doc-To-Help features Microsoft® Office 2010 style ribbons. These ribbons are available not only in Doc-To-Help itself, but there are also special Doc-To-Help ribbons that are accessible in Word templates and documents associated with a Doc-To-Help project.

Ribbons in Doc-To-Help

Ribbons in Doc-To-Help

Doc-To-Help has multiple ribbons that are used for various purposes. The ribbon layout consists of a File drop-down with common tasks such as creating and opening new projects, as well as accessing unique tools and options. This is followed by the Home ribbon, which is the default, and other ribbons that are related to various features in .

Each ribbon is divided into logical groups. Many dialogs can be opened directly from ribbon groups using dialog launchers. (The dialog launchers are the small arrows on the bottom-right of specific ribbon groups.)

You can minimize the Ribbons using the Quick Access Toolbar option, which can be accessed from the down arrow located above the ribbons. See Quick Access Toolbar.

For other navigation methods, see Touring the Workspace.

Team Authoring

Team authoring is a source control feature that makes it possible for multiple authors to work together on a single project. Authors work on their own local copy of a project on their machine (called the working copy), while the master project (or the team project) is located on the organization's network or on a web server. Although the same project is made available to the entire team, one author's changes will not be overwritten by another author's changes. See Team Authoring. See the online Help or the Managing Your Project Guide.

The following commands are available in the Doc-To-Help ribbon when it is opened in Full mode (see Dynamic Add-In Modes).

Open the project associated with the current document in Doc-To-Help.

Open the Doc-To-Help Project panel. You can use this panel to access some of Doc-To-Help's most frequently used functions without having to leave Word. This panel includes four window panes: Documents, Contents, Index and Groups, and Variables. See Doc-To-Help Project Panel.

Open the Doc-To-Help Build panel. This panel allows you to build, rebuild, view, and cancel builds directly from Word. You can also perform builds for specific targets by selecting the target from a drop-down. A progress bar and percentage indicator in the bottom right corner of the panel will also display build progress. Additionally, if there is a build error or unresolved link, you can view the errors from this panel. See Doc-To-Help Build Panel.

Find and open another Doc-To-Help project containing the current source document. If the Doc-To-Help Word add-in is currently running in Basic mode, you can use this button to find a project and switch to Full mode. See Finding Projects Containing a Source Document. Apply the Heading 1, 2, 3, 4, or Body Text style to the selected text in the source document.

Apply or clear a D2HML style. See About Doc-To-Help Markup Language. See the online Help or the Making It Look Good Guide.

Insert a note or image in the left margin of a printed target. The margin note can be displayed as a popup in online outputs if desired. See Creating Margin Notes. See the online Help or the Documents, Templates, and Content Guide.

In online outputs, use the Image Map Editor to create hyperlinks within an image. See Creating Image Maps. See the online Help or the Documents, Templates, and Content Guide.

You may insert multimedia (video and audio) into online outputs. See About Multimedia . See the online Help or the Documents, Templates, and Content Guide.

Insert HTML Help ActiveX Controls into your Word documents. See Inserting an HTML Help ActiveX Control. See the online Help or the Documents, Templates, and Content Guide.

Insert cross-references that will appear as page numbers in printed outputs and hyperlinks in online outputs. See Inserting Cross-References. See the online Help or the Documents, Templates, and Content Guide.
