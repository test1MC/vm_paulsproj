Let's say that every time you create a new report in Analyzer, you want the report to include statistics, broken links, and absolute links. You may also want the reports to always show the headings in red font and the data in Verdana. Therefore, you create a report, selecting the check boxes for statistics, broken links, and absolute links. You also modify the style for the report so that the headings display in red, and the text displays in the font Verdana. When you are finished, you can save the report file as a template. In the future when you create new reports, you can select this report as the template for the new ones.

Now let's say that you create a new report and want it to include used condition tags in the output (in addition to statistics, broken links, and absolute links). You want the look of the report to have the familiar red headings and Verdana font for the data. When you create the new report, you select your template. Therefore, you do not need to worry about selecting statistics, broken links, and absolute links. You also do not need to worry about modifying the look of the report. Because you selected the template, all of those settings are already in place for you. The only adjustment you need to make is to select the check box for used condition tags. After that, you are ready to generate the new report.
