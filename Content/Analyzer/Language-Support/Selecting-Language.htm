Selecting a Language

You can use the following steps to select a language.

how to select a language for a specific topic

    • Ribbon Select View > File List.

    • Keyboard Shortcut Press CTRL+SHIFT+J on the keyboard.

    The File List window pane opens.

  2. Right-click on a topic and select Properties. The Properties dialog opens.
  3. Select the Language tab.
  4. From the list, select a language (those in bold font are supported in spell check).
