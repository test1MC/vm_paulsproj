Viewing Broken Bookmarks

 

how to view broken bookmarks

    • Ribbon In the Dynamic Reports section, select Links > Broken Bookmarks.

    • Analyzer Summary Window Pane Double-click the row that mentions broken bookmarks.

    The Broken Bookmarks window pane opens, listing all of the occurrences of absolute links in the project.

    • File
    • Folder
    • Link
    • Type
    • Link Text
    • Link Tag
    • Extension

how to fix broken bookmarks

  2. Redirect the link as necessary (e.g., right-click on the link and select Edit Hyperlink or Edit Cross-Reference).

    See the online Help in Flare for more information about creating and editing navigation links.

 notes

You can also see broken bookmarks in a file if you have the Link Viewer window pane open. See Viewing File Links.
