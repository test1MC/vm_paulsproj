Opening Reports

how to open a report file in the interface
  1. Open a project.
  2. In the Analysis section select Reports. The Reports window pane opens.
    • Double-click the report file that you want to open.
    • Select the report file that you want to open. In the local toolbar, click .

    The Report Editor opens, with the previous settings still in place. You can re-generate a report with those settings or use the Archive tab to open a report that you have saved previously.

 how to open a generated report in a browser window
  1. Generate a report. See Generating Reports.
  2. In the Report Editor, do one of the following.
    • Select the General tab. Click the Open in Browser button.
    • If you have previously saved a generated report in the archive, select the Archive tab. Select the report and click the Open in Browser button.

    After opening a report in a browser window, you can print it.
