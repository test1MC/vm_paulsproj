Setting Color Options for Backup Differences

For more information see Viewing Differences in Backup Files and About Backups.

example how to set color options for backup file differences
  1.  In the Common section, select Backups. The Backups window pane opens.
  2. Select the file whose differences you want to see.
  3. In the local toolbar of the Backups window pane, click Differences. The Differences Editor opens to the right.
  4. In the local toolbar of the Differences Editor, click Options to open the File Differences dialog. See File Differences Dialog.

  5. In the File Differences dialog, set the color options for the Difference types in the list:

    1. Click the Text or Background cell to open a menu.
    2. In the menu, select Pick Color to open the Color Picker dialog.
    3. In the Color Picker, set the available options. See Color Picker Dialog.
  6. Click OK to close the Color Picker dialog.
  7. Click OK to close the File Difference dialog.
