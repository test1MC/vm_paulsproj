Viewing Files with Changes

how to view files with changes
  1. Open a project.

  2. In the Dynamic Reports section select More Reports > Files With Changes.

    The Files With Changes window pane opens, displaying all topics in your project where changes have been inserted.

    • File
    • Title
    • Folder
    • Change Displays the type of change that was made.
    • Initials Displays the initials of the individual who made the change.
    • Creator Displays the user name of the individual who made the change.
    • Timestamp Displays the date and time when the change was made.

  5. If you double-click a file in the list, you will see a message stating that the file has unresolved changes in it. Click Yes if you want to open the file in Flare to accept or reject the changes.
