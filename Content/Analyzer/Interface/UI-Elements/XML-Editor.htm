XML Editor

In the Analyzer interface, you can open a file in the XML Editor. For example, if you are working on fixing broken links, you can double-click a row in the Broken Links window pane and the related topic opens in the XML Editor. You can then address the issue in the editor, rather than having to open the file in the Flare project.

how to open the editor
  1. Open one of the window panes that allows you to take action on existing content (e.g., Broken Links window pane, Snippet Suggestions window pane, Variable Suggestions window pane).
  2. Double-click a row in the window pane.
