Frame Properties Dialog

This dialog is used to edit frames that you add to a page layout. See Editing Pages and Frames.

Following are the primary tasks that you can perform in the Frame Properties dialog.

Tasks Associated with Frame Properties
  • Aligning Frames See Aligning Page Layout Frames.
  • Aligning Text See Aligning Text in Frames.
  • Anchors See Setting Frame Anchors.
  • Background See Specifying a Frame Background.
  • Borders See Specifying Frame Borders.
  • Columns See Creating Columns.
  • Floating and Sinking See Floating and Sinking Frames.
  • Frame Flow (Type) See Specifying Frame Flow.

  • Heading and Running Head Variables See Inserting Heading Variables Into Frames and Inserting Running Head Variables into Frames.

  • Padding See Specifying Frame Padding.
  • Page Numbers See Inserting Page Numbers into Frames.
  • Removing Frames See Removing Page Layout Frames.
  • Rotating Frames See Rotating Frames.
  • Size See Specifying Frame Size.
  • Text and Content—Add to Frames Simply by selecting a frame and pressing F2, you can open the Frame Contents window pane, which is used as a miniature XML Editor for adding content to a frame. See Adding Text and Content to Frames.
