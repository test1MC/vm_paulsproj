Properties Dialog (Topics)

Following are the primary tasks performed in this dialog.

Tasks Associated with Topic Properties
  • Condition Tags You can apply condition tags to the topic file.
  • Description 
  • File Tags You can associate file tags with the topics.
  • Language You can select a language to be used for the topic.
  • Search (Include/Exclude) 
  • Snippet Conditions You can create snippet conditions in a topic.
  • Snippet Variables When you set topic-level snippet variable definitions in the Topic Properties dialog, variables in snippets in that topic will use that definition.
  • Style Class
  • Title You can change the name of the topic title.
