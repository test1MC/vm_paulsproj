Index Window Pane

  • Ribbon In the Reference section select Index Window.
  • Keyboard Shortcut Press F9.
 what you can do in this window pane

Following are the primary tasks that you can perform in this window pane.

  • Create Index Keywords and Subkeywords There are two ways to link a topic to an index keyword using this window pane. First, you can open the topic (or a snippet), place your cursor where you want to insert the keyword, and use the top area of the Index window pane to create the keyword. Second, you can also use the bottom area of the Index window pane to add index keywords or subkeywords) and assign topics to them.
