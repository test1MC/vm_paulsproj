Project Properties Dialog

This dialog is used to see basic information (e.g., location of files, creation date) and to make property changes for your Flare project in the following ways.

  • Master Stylesheet Read more…  
  • Language Read more… You can select a language to use for the project.
  • Source Control Read more… You can select several options related to source control. The options shown are different, depending on which source control provider your project is bound to.
    • Check In Files Read more… You will see this option if your project is bound to Microsoft Team Foundation Server or Microsoft Virtual SourceSafe.
    • Commit Read More…

    • Submit Read More… You will see this option if your project is bound to Perforce.

    • Check Out Files Read more… You will see this option if your project is bound to Microsoft Team Foundation Server, Microsoft Virtual SourceSafe, or Perforce.
    • Lock Files Read more…
    • Unlock Files Read more…
    • Get Latest Version Read more… You will see this option if your project is bound to Microsoft Team Foundation Server, Microsoft Virtual SourceSafe, or Perforce.

    • Update Read More…

    • Undo Check Out Read more… You will see this option if your project is bound to Microsoft Team Foundation Server or Microsoft Virtual SourceSafe. See Undoing a Checkout of Source Control Files.
    • Revert Read more… You will see this option if your project is bound to Git, Perforce, or Subversion.

    • View History Read more…
    • Request Check In Read more… You will see this option if your project is bound to Microsoft Team Foundation Server or Microsoft Visual SourceSafe.
