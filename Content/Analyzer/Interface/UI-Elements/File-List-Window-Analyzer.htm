File List Window Pane

This window pane lets you view all content files in the Flare project being analyzed. Perhaps the most important thing you can do in the File List window pane in (besides quickly finding any content file in the project) is to send files to the File Issues Viewer, which lets you see any issues associated with those specific files.

  • Ribbon In the Analysis section, select File List.

  • Keyboard Shortcut Press CTRL+SHIFT+J.
 what you can do in this window pane

Following are the primary tasks that you can perform in the File List window pane, depending on the file type.

  • Column Information (Customize) Read more… See Customizing Column Information.
  • Condition Tags (Apply) Read more… You can apply condition tags to files. To do this right-click on the file, select Properties, and use the Conditional Text tab. See Applying Conditions to Content.
  • File Issues Viewer (Send Files) Read more… You can send selected files to the File Issues Viewer in order to see any issues associated with them. To do this click in the local toolbar or right-click on the file and select Send to File Issues Viewer. See Viewing Issues for Specific Files.
  • File Tags (Associate) Read more… You can associate tags with files. To do this right-click on the file, select Properties, and use the File Tags tab. See Associating Tags with Files.
  • Flare (Send) Read more… You can send a file to Flare so you can work on it in that application. To do this click on the file and select File > Send to Flare.
  • Language (Select) Read more… You can select a language to be used for a file. To do this right-click on the file, select Properties, and use the Language tab. See Selecting a Language.
  • Read Only Read more… You can mark files as "Read Only" so that they cannot be modified. To do this right-click on the file, select Properties, and use the General tab.
  • Snippet Conditions (Apply) Read more… You can apply snippet conditions to a file. To do this right-click on the file, select Properties, and use the Snippet Conditions tab.
  • Source Control (View Checked Out Files) Read more… If you are using Git or Subversion, you will see files you have modified. To do this right-click on the file and select Source Control > Pending Changes. See Viewing Files that are Checked Out or Modified.
  • Styles/Topics (Apply) Read more… You can apply style classes to topics. To do this right-click on the file, select Properties, and use the Topic Properties tab. See Setting the HTML Style for Topics.
  • XML (Convert) Read more… You can convert a non-XML file in to XML. To do this right-click on the file and select Convert to XML.
  • XML (Validate) Read more… You can fix invalid XML (e.g., unclosed tags) in a file. To do this right-click on the file and select Validate XML. See Validating XML Files.
