Capture Editor

When you capture, create, or open an image, it displays in the Capture Editor. You can then use the tools in the Capture Editor to add objects such as callouts, shapes, or effects to the image in order to change its appearance.

You can have as many images open in Capture at the same time. Each image is displayed in its own instance of the Capture Editor. When you capture a new image or open an existing one, this editor opens automatically.

how to open this editor
  • When you capture a new image or open an existing one, this editor opens automatically.
 what you can do in this editor

You can accomplish many things in this editor. Some of the primary tasks that you can perform are listed below.

  • Anchors (Set) Read more… See Setting Object Anchors.
  • Aligning Objects Read more… After objects are added, you can align them on the image. There are two ways to align objects: (1) in relation to one of the objects, or (2) on the canvas. See Aligning Objects.
  • Arrow (Set) Read more… You can change many aspects of an arrow to change its shape. See Setting the Shape of an Arrow.
  • Blur Effect (Add) Read more… See Adding Blur Effects.
  • Blur-Inside Effect (Add) Read more… See Adding Blur-Inside Effects.
  • Border (Add) Read more… See Adding Borders.
  • Color—Image Background (Set) Read more… From time to time, you may find it necessary or useful to select a specific color for an image's background. By default, the image's background is transparent (gray and white checks). See Setting the Color for an Image Background.
  • Color—Objects (Set) Read more… See Setting the Color for an Object.
  • Conditions (Apply) Read more… See Applying Conditions.
  • Conditions (Preview) Read more… See Previewing Conditions.
  • Cropping Images Read more… See Cropping Images.
  • Floating and Sinking Objects Read more… When you add an object, it is placed on its own layer. And each time you add a new object, it is placed on the top layer with the previous objects on layers beneath it. If necessary, you can "float" objects that are on lower layers to bring them forward, and you can "sink" objects to send them backward. See Floating and Sinking Objects.
  • Font Properties (Set) Read more… See Setting Font Properties for Text.
  • Gray Scale Effect (Add) Read more… See Adding Gray Scale Effects.
  • Grouping Objects Read more… You can group multiple objects into an object group, which then acts as a single object. And at any point, you can ungroup the objects so that they are separate once again. See Grouping Objects.
  • Hiding Objects Read more… You can hide an object in an image so that it cannot be seen. Later, you can always bring the object back into view. See Hiding Objects.
  • Language (Select) Read more… See Selecting a Language.
  • Lines (Add) Read more… Capture has a line tool that lets you create different types of lines with one or more line segments. You can create several types of lines, including polylines, curved lines, and zig zag lines. You can also specify whether the line should include arrows, a shadow, or other properties. See Adding Lines.
  • Locking Objects Read more… You can lock an object in place so that it cannot be moved on the image. Later, you can always unlock the object if necessary. See Locking Objects.
  • Moving Objects Read more… After an object is added, you can move it around the image to place it just where you need it. You can do this by dragging the object. See Moving Objects Around.
  • Padding (Add) Read more… See Adding Padding and Adding Padding to Objects.
  • Profile (Apply) Read more… See Applying Profiles to Images.
  • Profile (Detach) Read more… See Detaching Profiles from Images.
  • Resizing Images Read more… You can resize images in a different ways. You can resize the entire image, including all objects on it. Alternatively, you can resize just the image background. See Resizing Images.
  • Resizing Objects Read more… After an object is added to an image, you can resize it. You can do this by dragging the edges of the object or by automatically resizing a group of objects so that the height and/or width are the same. See Resizing Objects.
  • Resizing the Canvas Read more… See Resizing the Canvas.
  • Rotating Objects Read more… You can rotate objects after adding them to images. See Rotating Objects.
  • Shading Effect (Add) Read more… See Adding Shading Effects.
  • Shadow Effect (Add) Read more… See Editing Shadow Effects.
  • Shapes (Add) Read more… There are various shapes that you can add to an image, such as bubbles, loops, arrows, polygons, ovals, and rectangles. See Adding Shapes to Images.
  • Text (Add) Read more… See Adding Text.
  • Torn Edge Effect (Add) Read more… You can easily add an effect around an image or an image object to give it the appearance of having a torn edge. See Adding Torn Edge Effects.
  • Zoom Effect (Add) Read more… See Adding Zoom Effects.
