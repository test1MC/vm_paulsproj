Steps for Using Condition Tags

Following are the basic steps involved with condition tags.

  1. (Optional) Link to Flare Project Read more… If you want to use existing condition tags from a MadCap Flare project, you can link your image to that project. There are two ways to do this. The easiest method is to simply insert the image into a Flare project. Then when you open that image in to edit it, the condition tags from the project are already available. The second method is to manually link the image to the project from within . For the first method, see the online Help provided with Flare. For the second method, see Linking to Flare Projects.
  2. (Optional) Create Condition Tags Read more… See Creating Condition Tags.
  3. Apply Condition Tags Read more… See Applying Conditions. For more information see Saving an Image and the examples below.
  4. Associate Condition Tags with Images Read more… See Associating Condition Tags with Images.
  5. (Optional) Preview Conditions Read more… See Previewing Conditions.

Example—Linking to Flare Project

Let's say you have a Flare project and want to create one set of output for advanced users and another set for beginners. You might create one condition tag and name it Advanced, while naming a second condition tag Beginners.

Now you open and use it to create a new image. Suppose you want to use this image in your Flare project, but you want it to be included only in the output intended for advanced users.

One way to accomplish this is to add the image to your Flare project and then apply the condition tag to the image file once it is inside the project. However, you can do the same thing before the image is even added to the project. You can apply the condition tag to the image while it is still inside .

First, you would double-click on the image. On the Project tab, you use the button to find and link the image to the Flare project.

You then save your changes and close the image file.

Next, you reopen the image file and double-click it again. This time you select the Conditional Text tab. All of the condition tags from the linked Flare project are listed. Because you want this particular image to be used for advanced users rather than beginners, you select the appropriate Include and Exclude check boxes.

After bringing the image into your Flare project, you will notice that the condition tags are already applied to the image file.
