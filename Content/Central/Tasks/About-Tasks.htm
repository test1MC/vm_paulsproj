Four Views—Task Board, Calendar, Archive, Backlog

Four Views—Task Board, Calendar, Archive, Backlog

The Tasks page is a single framework with multiple views, which can be accessed on the right side of the interface. The different views are:

  • Task Board (default view)
  • Calendar
  • Archive
  • Backlog

Task Board

Task Board

The Task Board view is similar to a Kanban board. It consists of three preset milestones: To Do, In Progress, and Completed.

When you create a new task, it is placed by default in the “To Do” column. As the task progresses, you can click and drag it, moving it to “In Progress,” and finally to “Completed.”

Calendar

Calendar

The Calendar view displays tasks in a calendar format for the day, week, and month. This view is useful for seeing when tasks begin and end in relation to one another, based on their start and due dates. Tasks that do not have start and due dates are shown as all day events in the day and week views. Under the calendar, the tasks are listed in a grid.

When you select a task, the Task Details dialog opens so that you can make adjustments.

Archive

Archive

The Archive view displays tasks that have been completed.

In order to archive tasks, they must first be moved to the Completed milestone. From there, they can be moved to the archive.

Backlog

Backlog

The Backlog view displays tasks that are not yet complete and need to be removed from the work flow. The idea is that backlogged tasks will be revisited in the future so that they can be completed.

Only tasks in the "To Do" or "In Progress" milestone columns can be moved to the backlog.

Activity Tracker

Activity Tracker

There is a tracker on the right side of the Tasks page showing the most recent task activities.
