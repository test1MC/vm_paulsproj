Changing Your Password

How to Change Your Password
    • Top Navigation At the top of Central, click your name and select User Settings.
    • Users Page Click . The Users page opens. Hover over your avatar or name, and from the user card select Edit Profile.

    The User Settings dialog opens.

  2. On the Profile tab, click Change Password. The Change Password dialog opens.
  3. Complete the Current Password field.
  4. Complete the New Password field.
  5. Complete the Confirm New Password field.
  6. Click Save.
  7. In the Settings dialog, click Save.
