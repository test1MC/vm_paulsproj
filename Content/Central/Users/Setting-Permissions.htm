Setting User Permissions

If you attempt to perform a task for which you do not have permission, you will be informed of this. You can then contact an administrator (identified by a crown on the user's avatar) to request permission for that task.

see the movie

How to Set Permissions
  1. Click . The Users page opens.
  2. Find the user in the grid, and click in an empty space so that the entire row is highlighted.
  3. In the toolbar click . The Set Permissions dialog opens.

  4. From the Level field, select either Global Permissions (All Projects) or choose the name of a specific project.

    Note

  5. Select any of the check boxes to give the user the appropriate permissions in the system.

    Permissions
  6. Click Save and Finish.
