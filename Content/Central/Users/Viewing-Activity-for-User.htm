Viewing the Activity for a User

How to View the Activity for a User
  1. Click . The Users page opens.
  2. Find the user in the grid.
    • User Card Hover over the user’s name until the user card pops up. Then click Activity.
    • User Profile Dialog Double-click an empty space in the row. The User Profile dialog opens. Click Activity.

    A dialog shows the most recent activity for that user.

  4. When you are finished, click Close.
