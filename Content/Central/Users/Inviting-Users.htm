How to Invite a Single User
  1. Click . The Users page opens.
  2. In the toolbar click . The Invite User dialog opens.
  3. Complete the name and email address fields.
  4. Click the arrow next to Set Permissions. The area expands, showing permission check boxes.

  5. Select any of the check boxes to give the user the appropriate permissions in the system.

    Permissions
  6. Click Send Invitation. If the invitation is successful, an email is sent to the user. The user must click the link in the email to activate the account. Until the user activates the account, he or she is listed as "Invited" in the system.

If no user seats are remaining, you will be informed. You must purchase more user seats before adding the new one. See Purchasing User Seats and Storage Space.

Alternatively, you can deactivate an existing user to free up a user seat. See Deactivating Users.

For more information, see the online Help.

 How to Invite Multiple Users in Bulk

  1. Create, or make sure you have access to, a comma-separated value (CSV) file containing the information for the users you want to add. The CSV file should have three columns, with the following in the top cells of each column:

    • Email
    • FirstName
    • LastName
  2. Click . The Users page opens.
  3. In the toolbar click . The Invite User dialog opens.
  4. Click Bulk Invite. The Bulk Invite dialog opens.
  5. Click Browse.
  6. Find and select the CSV file. Then click Open.
  7. Click the arrow next to Set Permissions. The area expands, showing permission check boxes.

  8. Select any of the check boxes to give the users the appropriate permissions in the system. All of the users in the CSV file will receive the same permissions that you choose. However, you can always edit the permissions for a particular user later (see Setting User Permissions).

    Permissions
  9. Click Upload File. The dialog indicates the new users that will be invited. Also, it will show if any of the selected users are already on the license, as well as any invalid emails that are found in the CSV file.
  10. Click Accept & Send Invitation. If the invitations are successful, an email is sent to each user. The user must click the link in the email to activate the account. Until the user activates the account, he or she is listed as "Invited" in the system.
 Note

You can change the user's permissions any time. In addition, you can set permissions globally for all projects or separately for individual projects. See Setting User Permissions.
