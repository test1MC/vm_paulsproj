How to Set Up Slack Integration
  1. In Slack, make sure you have set up a Slack account and created channels that you want to integrate with Central.

  2. At the top of Central, click your name and select Server Settings.

    The Server Settings dialog opens.

  3. Select the Slack tab.

  4. In the Channels section, click .

    Slack opens in your browser.

  5. Choose your Slack team. And then in the Post to field, select a channel that you want to integrate.
  6. Click Authorize.

  7. Back in Central, a message indicates that addition was successful. Click OK.

    The channel is now shown in the grid.

  8. Repeat these steps if you want to add more of your Slack channels to the grid.

  9. With your channel added to the tab, you’re ready to connect it to any of Central’s notifications. In the bottom area of the tab, click .

    The Add Channel Notification dialog opens.

  10. In the Name field, provide a name for the channel notification. It can be the same name as the Slack channel you want to hook it to, but this isn’t necessary.

  11. From the Channel field, select the Slack channel that you want to associate with the notifications.

  12. From the Activity Type field, select one of the alert categories—Builds, Projects, Tasks, Teams, Users. Notice that these are the same groups available when you set up personal notifications in your user settings.

  13. Depending on the activity type you choose, additional fields are displayed in the area below. Complete the fields as necessary to choose the type of notifications you want to be associated with the Slack channel.

    Builds

    1. From the Project field, select either All Projects or choose a specific Flare project whose builds you want to integrate.

    2. (Optional) If you chose a specific project, you can make a selection in the Target field. Select either All Targets or choose a specific target whose builds you want to integrate.

    3. From the Activity field, select any of the build activities to include in the integration.

      Field

      Description

      Completed

      See Running and Scheduling Builds.

      Set as Live

      See Setting Builds to Live or Keep.

      Started

      See Running and Scheduling Builds.

      Stopped

      See Running and Scheduling Builds.
    Projects

    1. From the Project field, select either All Projects or choose a specific Flare project you want to integrate.

    2. From the Activity field, select any of the project activities to include in the integration.

      Field

      Description

      Deleted

      See Deleting Projects.

      Profile Changed

      See Editing Project Profiles.

      Status Changed

      See Changing the Status of a Project.
    Tasks
    1. From the Tasks field, select either a specific task or choose one of the following:
      • All Tasks
      • All Tasks WITHOUT a Project
      • All Tasks for a Specific Project

    2. (Optional) If you selected “All Tasks for a Specific Project,” click in the Project field and choose the project with related tasks that you want to integrate.

    3. From the Activity field, select any of the tasks activities to include in the integration.

      Field

      Description

      Comments or File Actions

      See Adding Comments to Tasks and Editing Tasks.

      Deleted

      See Deleting Tasks.

      Edited

      See Editing Tasks.

      New/Assigned

      See Creating Tasks.

      Status Changed (Moved)

      See Moving Tasks.
    Teams

    1. From the Team field, select either All Teams or choose a specific team you want to integrate.

    2. From the Activity field, select any of the team activities to include in the integration.

      Field

      Description

      Added/Removed from a Project

      See Editing Teams.

      Deleted

      See Deleting Teams.

      Edited

      See Editing Teams.
    Users

    1. From the User field, select either All Users or choose a specific user you want to integrate.

    2. From the Activity field, select any of the user activities to include in the integration.

      Field

      Description

      Added/Removed from a Project

      See Assigning Users to Teams and Projects.

      Added/Removed from a Team

      See Assigning Users to Teams and Projects.

      Permissions Changed

      See Setting User Permissions.

  14. In the Server Settings dialog, a row is added for the new channel notification.

  16. (Optional) You can perform an action in Central that you have integrated with the Slack channel (e.g., build a target). You should see the notification added to your Slack channel.

 Notes

Keep in mind that the notification UI in Central is unique to each user (see Editing Your Account Settings). On the other hand, when you set up Slack integration, it is created for the entire license; therefore, other users can tap in to that channel notification to receive the same alerts via Slack.

If you encounter problems or need help with your Slack integration, please contact support. See http://www.madcapsoftware.com/support/contactoptions.aspx.
