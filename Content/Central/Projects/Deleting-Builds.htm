Deleting Builds

You can delete a single build for a target, or you can delete multiple builds at the same time.

How to Delete a single Build
  4. In the grid, click the build that you want to delete.
  5. In the toolbar, click .
  6. In the confirmation dialog, click OK.
 How to Delete Multiple Builds
  3. (Optional) You only need to do this if you plan to delete multiple builds for a certain target. Alternatively, you have the option of deleting multiple builds for all targets throughout the project.
  4. In the toolbar, click .
  5. From the drop-down, select one of the following options:
    • Delete all stopped builds in the selected target
    • Delete all failed builds in the selected target
    • Delete all builds in the selected target
    • Delete all stopped builds in project
    • Delete all failed builds in project
    • Delete all builds in project
  6. In the confirmation dialog, click OK.
 Note

You cannot delete builds that are set to “Live” or “Keep.”
