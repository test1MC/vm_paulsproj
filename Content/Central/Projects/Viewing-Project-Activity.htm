How to View Project Activity

  2. Click Activity.

    The Recent Activity dialog opens.

  3. (Optional) You can click any blue information link to open it. For certain items (e.g., users, projects), you can also hover over them to display the relevant card (e.g., user card, project card). You can also click Load More Activity if you want to see more entries.

  4. When finished, click Close.
