Main Navigation Framework

Top Navigation

Opens the Home page, where you can add widgets to the dashboard. See About Dashboards and Widgets.

Drop-down field showing your avatar and user name. If you do not have an avatar set, your initials will be shown. From the drop-down you can choose the following:

  • User Settings See Setting Your Profile and Notifications.
  • Server Settings See About Server Management and Purchasing.
  • Sign Out Lets you log out of the system.

Displays the label for your license key. All of the information you see throughout the interface is tied to this license.

This field displays only if you have access to more than one Central license. It lets you switch from one license to the other without having to sign out. If you belong to only one license, you will not see this field.

Shows alerts (indicated by a number) for new and recent notifications in the system. Clicking the icon shows recent notifications. See Setting Your Profile and Notifications.

You can click View All Notifications to open the Notification Center.

After you close the Notification Center, your message alert count is reset on the icons.

Shows alerts (indicated by a number) for new messages.

Clicking the icon opens the Message Center (see Using the Message Center). This interface provides a chat platform for all users in the system under the selected license.

After you close the Message Center, the alert count is reset.

From the drop-down you can choose the following:

  • Documentation Opens the online Help in a separate browser tab.
  • Submit Feedback Opens a web page in a separate browser tab, where you can submit bugs and provide other feedback.
Left Navigation

Opens the Home page dashboard, where you can add widgets pointing to various information in the system. See About Dashboards and Widgets.

Opens the Projects page, where you can manage uploaded Flare projects and builds. See About Projects.

Opens the Tasks page, where you can create, edit, and view tasks. See About Tasks.

Opens the Teams page, where you can create and manage teams. See About Teams.

Opens the Users page, where you can manage and view users on the license. See About Users.
About Box

Clicking the Central logo on the bottom-left of the interface opens the About box, which provides basic information about MadCap Central.
