Filtering Widgets

How to Filter a Widget

    • Home Page Click . The Home page opens.

      OR

    • Project Dashboard Click . The Projects page opens. Click a project name. The Project dashboard opens.

  2. In the widget you want to filter, click in the upper-right corner.

  3. In the menu, do the following, depending on the filter you want to use.

    Filter by Project
    1. Click Filters > Select Project.
    2. In the dialog, click to the left of the project you want to filter by, so that a displays.
    3. Click Save and Finish. The widget updates to show information for the selected project.

    You can filter the following widgets by project:

    • Builds Activity
    • Live Builds
    • Project Properties
    • Build History
    • Task Calendar
    • Task Summary

    You can add projects to the Project Bookmarks widget from the widget menu . However, you cannot filter the widget.

    Example

    You have several projects in Central. Usually, you like to see the build activity for all of your projects in the Builds Activity widget, so you do not filter it.

    However, you have a big release coming up for one of your projects so you filter the Builds Activity widget to only show the builds for that project. This way, you can keep an eye on the builds for that project to make sure everything is on track for your release.

         Filter by Target

    In the Build History widget, you can filter by project, and then by a target within that project.

    1. After you have already filtered the widget for a project, click Filters > Select Target.
    2. In the dialog, click to the left of the target you want to filter by, so that a displays.
    3. Click Save and Finish. The widget updates to show information for the selected target.
         Filter by User
    1. Click Filters > Select User.
    2. In the dialog, click to the left of the user you want to filter by, so that a displays.
    3. Click Save and Finish. The widget updates to show information for the selected user.

    You can filter the following widgets by user:

    • Task Calendar
    • Task Summary

    Example

         Filter by Priority

    In the Task Calendar widget, you can filter by priority level.

    1. Click Filters > Select Priority.
    2. In the dialog, click to the left of the priority level(s) you want to filter by, so that a displays.
    3. Click Save and Finish. The widget updates to show information for the selected priority level.

    Example

    You want to filter the Task Calendar widget by priority. You have several tasks due soon, but a few are very urgent and some are less pressing. You decide to duplicate the Task Calendar widget so you have three copies.

    Then you create priority filters for each one: one by High priority, one by Medium priority, and one by Low priority.

    This lets you see all of your tasks, but separated by priority so you can easily decide which ones to tackle first.

         Filter by Dates

    In the Task Calendar widget, you can filter by both a date range and a due date.

    1. Click Filters > Select Dates.
    2. (Optional) In the Start Date and End Date fields, enter the date range you want to filter by (MM/DD/YYYY format), or click to select a date from a calendar.
    3. (Optional) In the grid, click to the left of the date options you want to filter by, so that a displays.
    4. Click Save and Finish. The widget updates to show information for the selected dates.

    Example

    This is what your Task Calendar looks like right now. You have a lot of tasks due... and a lot of tasks overdue.

    You want to filter the widget to see only the tasks that are overdue. So you filter for overdue tasks.

    You realize that you have more overdue tasks than you anticipated.

    You click and reopen the Date filter (Filters > Select Dates), then use the date range filter to narrow down the date range to show only overdue tasks that were due last week, since they were most recent. Then you can focus on those tasks first.

         Filter by State

    In the Builds Activity widget, you can filter by build state.

    1. Click Filters, then hover over State.

    2. From the submenu, select the states you want to filter by, so that a check mark is next to each one. After you select a state, you will need to navigate to the submenu again to select more.

      • Failed The build was unsuccessful.
      • Complete The build was successful.
      • Canceled The build was terminated before it could finish.
      • Building The build is in the process of generating.
      • Waiting The build is scheduled but has not yet started.
      • Getting files The build is in the process of retrieving files before generating.
      • Publishing The build is finished generating and is now in the process of being published to the “Live” URL.

    Example

    You have built output several times in the last few weeks. Most of the builds were successful, but a few builds failed.

    You filter the Builds Activity widget by state so you only see the builds that have failed. This can help you figure out what might have caused the build to fail.
  4. (Optional) To clear the filters, click , then select Clear Filters.

Example—Multiple Filters

Before filtering the Task Calendar widget, you can see dozens of tasks on your calendar. It is confusing because the due dates are scattered over several months, many projects, and many users. Additionally, some tasks are urgent and some are low priority.

You know that you have a few urgent tasks due for a project you are working on, so you decide to apply several filters to your Task Calendar widget. First, you filter by user, so you only see your tasks.

However, there are still too many tasks on your calendar, so you decide to filter by project. This way you only see the tasks due for the project you are working on.

This really helps narrow down the tasks, but you want to focus on the most urgent tasks first, so you filter again to only show high priority tasks.

Now your calendar only shows a handful of tasks. Finally, you decide to filter by date, so you know which tasks to focus on first. You use the date filter to show just the tasks due in the next two weeks.

With all of these filters applied, you only see a few tasks on your calendar and you can focus on the most important and urgent tasks for your project.
