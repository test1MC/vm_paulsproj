How to Open the Message Center

You can open the Message Center in a few different ways.

  • At the top of the interface, click .

  • When viewing a user's profile, click Message.

  • When viewing a team's profile, click Send Message.

  • When viewing a user card, team card, or project card, click Message.

 How to Send a New Message
  1. In the Message Center, click New Message. An address field appears on the right side of the Message Center.
  2. In the To: field, enter the name of the user or team you want to message. As you type, users and teams whose names match your entry will appear below the field. Select the user or team you want to add.
  3. (Optional) If you want to message more than one user or team, continue adding names in the To: field.
  4. In the message box at the bottom of the pane, enter your new message.
  5. Click Send. (If you want to use the Enter key to send the message, select Press Enter To Send, then press Enter.)
  6. When you are done sending messages, click Finish to close the Message Center.
 How to View Messages

  1. On the left side of the Message Center, select how you want to display your messages.

    • Inbox Displays all messages.
    • Unread Only Displays only unread messages. Unread messages are shown in bold and display an unread count.
  2. Select the message you want to view. The messages in the conversation appear on the right side of the Message Center.

  3. (Optional) Click Delete next to any message to delete it.

    Deleting a message does not affect any other user's view of the message. You only delete your copy of the message.

  4. (Optional) Hover over any user or team in the message list and click to delete the entire conversation.
  5. (Optional) Click to refresh the message list.
 How to Search The Message Center
  • In the search box at the top of the Message Center, enter text and press Enter. The Message Center will search for the user name or for the text that you entered.
 How to Reply to a Message
  1. In the Message Center, select the message you want to reply to.
  2. In the message box at the bottom of the pane, enter your new message.
  3. Click Reply. (If you want to use the Enter key to send the message, select Press Enter To Send, then press Enter
  4. When you are done sending messages, click Finish to close the Message Center.
